1. Wprowadzenie

Interface pyplot biblioteki matplotlib zawiera szereg funkcji, umoliwiajcych stworzenie zarwno podstawowych, jak i zoonych wykresw. Podstawowe elementy, ktre mog zosta zawarte na wykresie wprowadzone zostay w wiczeniu (Pandas cz.2) - warto jednak zaznajomi si ze wszystkimi moliwymi skadowymi wykresu/ryciny oraz pozna ich angielskie nazwy (m.in. figures, grid etc.), ktre s przydatne szczeglne podczas korzystania z parametrw funkcji.

Ponisza grafika, stworzona z pomoc moduu matplotlib (https://matplotlib.org/stable/gallery/showcase/anatomy.html), przedstawia wszystkie te elementy:

2. Style wykresw

1. Wykres domylny - liniowy

Moemy stworzy wykresy przekazujc dwa wektory wartoci najpierw dla wektora x, nastpnie y. Dodatkowo moemy przekaza parametr w postaci wartoci typu string, ktry okreli styl wykresu, np. 'ro-'.
Wszelkie moliwoci mona sprawdzi na stronie dokumentacji: https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.plot.html

In[13]:
import matplotlib.pyplot as plt

plt.show()

Za pomoc pojedynczego wywoania funkcji plot() moemy wygenerowa wiele wykresw na jednym "ptnie" (ang. canvas) kadorazowo podajc niezbdne wartoci: wartoci dla osi x, wartoci dla osi y, styl wykresu

3. Podwykresy

Podwykresy pozwalaj na umieszczanie na jednym ptnie wielu wykresw.
Podajemy wymiary, czyli liczb wierszy oraz liczb kolumn, w ktrych umieszczone zostan podwykresy. Suy do tego funkcja subplot, ktra przyjmuje 3 argumenty (nrows, ncols, index).
Odpowiednio jest to: ilo wierszy, ilo kolumn oraz indeks definiowanego wykresu (indeksy rozpoczynaj si od 1 i kocz na nrows * ncols).

In[40]:
import numpy as np
import matplotlib.pyplot as plt


x1 = np.arange(0.0, 2.0, 0.02)
x2 = np.arange(0.0, 2.0, 0.02)

y1 = np.sin(2 * np.pi * x1)
y2 = np.cos(2 * np.pi * x2)


plt.subplot(2, 1, 1)
plt.plot(x1, y1, '-')
plt.title('Dwa podwykresy')
plt.ylabel('sin(x)')

plt.subplot(2, 1, 2)
plt.plot(x2, y2, 'r-')
plt.xlabel('x')
plt.ylabel('cos(x)')

plt.show()

Moemy rwnie pomija umieszczanie wykresu w danym miejscu siatki.

Popularnym typem wykresw dla zaprezentowania rozkadw prawdopodobiestwa s histogramy:

In[45]:
import numpy as np
import matplotlib.pyplot as plt

x = np.random.randn(10000)

plt.hist(x, bins=50, facecolor='g', alpha=0.75, density=True)

plt.xlabel('Wartoci')
plt.ylabel('Prawdopodobiestwo')
plt.title('Histogram')
# wywiatlanie siatki
plt.grid(True)
plt.show()
  • bins oznacza ilo koszy czyli supkw, do ktrych maj wpada wartoci z wektora x
  • facecolor oznacza kolor supkw
  • alpha to stopie przezroczystoci wykresu
  • density oznacza czy suma iloci zostanie znormalizowana do rozkadu prawdopodobiestwa, czyli przedzia 0,1

import numpy as np
import matplotlib.pyplot as plt

zawodnicy = ['Messi', 'Suarez', 'Dembele', 'Coutinho']

# pierwsza wersja wykresu

plt.title("Pierwsza wersja wykresu")
plt.legend(title='Zawodnicy')
plt.show()

# druga wersja wykresu
def prepare_label(pct, br):
    absolute = int(np.ceil(pct / 100. * np.sum(br)))
    return "{:.1f}% \n({}/{})".format(pct, absolute, sum(bramki))


wedges, texts, autotexts = plt.pie(bramki, labels=zawodnicy,
                                   autopct=lambda pct: prepare_label(pct, bramki), textprops=dict(color="black"))
plt.setp(autotexts, size=14, weight="bold")
plt.title("Druga wersja wykresu")
plt.legend(title='Zawodnicy')
