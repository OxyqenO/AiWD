1. Wprowadzenie¶

Interface pyplot biblioteki matplotlib zawiera szereg funkcji, umożliwiających stworzenie zarówno podstawowych, jak i złożonych wykresów. Podstawowe elementy, które mogą zostać zawarte na wykresie wprowadzone zostały w ćwiczeniu (Pandas cz.2) - warto jednak zaznajomić się ze wszystkimi możliwymi składowymi wykresu/ryciny oraz poznać ich angielskie nazwy (m.in. figures, grid etc.), które są przydatne szczególne podczas korzystania z parametrów funkcji.

Poniższa grafika, stworzona z pomocą modułu matplotlib (https://matplotlib.org/stable/gallery/showcase/anatomy.html), przedstawia wszystkie te elementy:

2. Style wykresów¶

1. Wykres domyślny - liniowy¶

Możemy stworzyć wykresy przekazując dwa wektory wartości najpierw dla wektora x, następnie y. Dodatkowo możemy przekazać parametr w postaci wartości typu string, który określi styl wykresu, np. 'ro-'.
Wszelkie możliwości można sprawdzić na stronie dokumentacji: https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.plot.html

3. Podwykresy¶

Podwykresy pozwalają na umieszczanie na jednym płótnie wielu wykresów.
Podajemy wymiary, czyli liczbę wierszy oraz liczbę kolumn, w których umieszczone zostaną podwykresy. Służy do tego funkcja subplot, która przyjmuje 3 argumenty (nrows, ncols, index).
Odpowiednio jest to: ilość wierszy, ilość kolumn oraz indeks definiowanego wykresu (indeksy rozpoczynają się od 1 i kończą na nrows * ncols).

In [40]:
import numpy as np
import matplotlib.pyplot as plt


x1 = np.arange(0.0, 2.0, 0.02)
x2 = np.arange(0.0, 2.0, 0.02)

y1 = np.sin(2 * np.pi * x1)
y2 = np.cos(2 * np.pi * x2)


plt.subplot(2, 1, 1)
plt.plot(x1, y1, '-')
plt.title('Dwa podwykresy')
plt.ylabel('sin(x)')

plt.subplot(2, 1, 2)
plt.plot(x2, y2, 'r-')
plt.xlabel('x')
plt.ylabel('cos(x)')

plt.show()

Popularnym typem wykresów dla zaprezentowania rozkładów prawdopodobieństwa są histogramy:

In [45]:
import numpy as np
import matplotlib.pyplot as plt

x = np.random.randn(10000)

plt.hist(x, bins=50, facecolor='g', alpha=0.75, density=True)

plt.xlabel('Wartości')
plt.ylabel('Prawdopodobieństwo')
plt.title('Histogram')
# wyświatlanie siatki
plt.grid(True)
plt.show()
  • bins oznacza ilość „koszy” czyli słupków, do których mają wpadać wartości z wektora x
  • facecolor oznacza kolor słupków
  • alpha to stopień przezroczystości wykresu
  • density oznacza czy suma ilości zostanie znormalizowana do rozkładu prawdopodobieństwa, czyli przedział 0,1

import numpy as np
import matplotlib.pyplot as plt

zawodnicy = ['Messi', 'Suarez', 'Dembele', 'Coutinho']
bramki = [48, 25, 13, 11]


# pierwsza wersja wykresu
wedges, texts, autotexts = plt.pie(bramki, labels=zawodnicy,
                                   autopct=lambda pct: "{:.1f}%".format(pct), textprops=dict(color="black"))
plt.setp(autotexts, size=14, weight="bold")
plt.title("Pierwsza wersja wykresu")
plt.legend(title='Zawodnicy')
plt.show()

# druga wersja wykresu
def prepare_label(pct, br):
    absolute = int(np.ceil(pct / 100. * np.sum(br)))
    return "{:.1f}% \n({}/{})".format(pct, absolute, sum(bramki))


wedges, texts, autotexts = plt.pie(bramki, labels=zawodnicy,
                                   autopct=lambda pct: prepare_label(pct, bramki), textprops=dict(color="black"))
plt.setp(autotexts, size=14, weight="bold")
plt.title("Druga wersja wykresu")
plt.legend(title='Zawodnicy')
